R语言的数据结构

1. 向量 Vector

向量是R语言的核心。一个向量内部的数据类型是单一的，不可以由整数元素和字符串混合构成。
在R语言中即使单个的数实际上也是由向量所表示的

6. 类 Class

R语言是一门面向对象的编程语言。每一个对象都有其类别属性，R语言的泛型函数可以根据变量的类别来执行。
通过class()查看变量的类



通过attributes()查看变量的属性，可以直接使用属性(变量)语句
